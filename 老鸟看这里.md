## 项目技术架构

### 技术栈说明
- 本项目是 **Streamlit** 项目
- 如果你希望以源码运行程序，你应该先确保安装了所有第三方库

## 运行方式

### 方式一：直接运行
- 直接运行 **"run.bat"** 文件

### 方式二：源码运行（推荐）
1. 打开命令行窗口
2. 切换到项目目录
3. 运行以下命令：
   ```bash
   streamlit run app.py
   ```

### 方式三：Python直接运行
- 也可以直接运行：
  ```bash
  python app.py
  ```

## GPU加速支持

### 当前设计考虑
- 本项目为了更高的适配性与低门槛，**没有使用支持GPU加速的Python框架**，如TensorFlow、PyTorch等

### GPU加速配置（可选）
- 如果你需要使用GPU加速，你应该**自行配置安装支持GPU加速的Python框架**，如TensorFlow、PyTorch等

### 运行模式说明
- 本项目支持 **CPU与GPU两种运行模式**
- 运行模式取决于你是否安装了支持GPU加速的Python框架

#### 自动检测机制
- **如果安装了支持GPU加速的Python框架**，本项目会自动检测并使用GPU加速
- **如果没有安装支持GPU加速的Python框架**，本项目会自动使用CPU运行
